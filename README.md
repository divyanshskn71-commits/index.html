# 💖 A Cute Animated Love Story Web 💖

Welcome to this beautiful, interactive, and animated mini-website! It tells a sweet story of a boy meeting a girl to deliver a secret romantic letter.

## ✨ Features (Kya Masala Hai Isme?)
- **Smooth Animations:** The boy walks automatically from outside the screen toward the girl.
- **Cute Dialogues:** Overthinking floating bubble text tracking the boy's inner thoughts.
- **Shy React Effect:** The girl turns around, changes her expression, and blushes when the boy reaches her.
- **Interactive Letter:** Clicking the letter opens a beautiful, customizable popup message.
- **Floating Hearts:** Endless romantic floating hearts fill the background upon opening the note!

## 💻 Tech Stack
- HTML5
- CSS3 (Keyframes & Animations)
- JavaScript (DOM Manipulation)
